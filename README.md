# 📄 رزومه آنلاین · توحید رجبعلی

این یه صفحه رزومه‌ست که با عشق و علاقه به توسعه‌دهنده‌ها تقدیم میشه.  
همه‌چیز توی یه فایل HTML جمع شده، با کمترین وابستگی و بیشترین سادگی.

---

## 🧩 اینجا چی می‌بینیم؟

- **معرفی کامل**: اطلاعات شخصی، تخصص‌ها، و یه توضیح کوتاه از خودم
- **مهارت‌های فنی**: دسته‌بندی شده تو چهار گروه backend, database, frontend, tools
- **زمینه‌های تجربه**: کارهایی که باهاشون حال کردم و توشون خوبم
- **پروژه‌های شاخص**: چند تا از کارهایی که انجام دادم یا روشون کار کردم
- **در حال یادگیری**: تکنولوژی‌های جدیدی که دارم می‌رم سراغشون

---

## 🛠️ چیزایی که استفاده کردم

- **Bootstrap 5.3** — برای ساختار و ریسپانسیو کردن صفحه
- **Font Awesome 6** — برای آیکون‌های قشنگ
- **فونت وزیر** — برای زیبایی و خوانایی بهتر در فارسی
- **یه کم خلاقیت شخصی** — برای اینکه از حالت خشک و رسمی دربیاد

---

## 🚀 چطوری اجراش کنم؟

اینقدر سادست که حتی نیازی به اجرا نداره!  
فقط کافیه فایل `index.html` رو با مرورگر باز کنی.

اگه دوست داری با یه سرور محلی ببینی:

```bash
# با پایتون
python3 -m http.server

# یا با VS Code و افزونه Live Server
# راست کلیک روی index.html > Open with Live Server
```

بعد برو به آدرس: [http://localhost:8000](http://localhost:8000)
