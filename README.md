# 📄 رزومه آنلاین | توحید رجبعلی

یک رزومه آنلاین تک‌صفحه‌ای برای معرفی مهارت‌ها، پروژه‌ها و زمینه‌های تخصصی در حوزه توسعه نرم‌افزار با تمرکز بر اکوسیستم .NET.

---

## معرفی

این پروژه با هدف ارائه یک رزومه سبک، سریع و بدون وابستگی‌های پیچیده طراحی شده است. تمام محتوای رزومه در یک فایل HTML قرار دارد و بدون نیاز به Build، Package Manager یا Backend قابل اجرا است.

---

## اجرا

فایل `index.html` را مستقیماً در مرورگر باز کنید.

یا در صورت نیاز از یک وب سرور محلی استفاده کنید:

```bash
python -m http.server
```

سپس:

```text
http://localhost:8000
```
